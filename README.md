Usage
-----

passport-verify-stub-relying-party uses `yarn` to manage dependencies. See https://yarnpkg.com/en/

Install the dependencies with:

```
yarn install
```

Run the tests with:

```
./pre-commit.sh
```

Start the application with:

```
./startup.sh
```
